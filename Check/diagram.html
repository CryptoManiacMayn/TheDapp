LIQUITY PROTOCOL: THE SOLUTION

Liquity = Decentralized Borrowing Protocol + Stability Pool Mechanism
  • Zero-interest loans in $LUSD (pegged to USD)
  • Collateralized by ETH (min. 110% ratio)
  • Decentralized frontends + immutable smart contracts
CORE MODULES
Troves (CDPs)
  • User-managed collateralized debt positions
  • Automated liquidations via Stability Pool
Stability Pool
  • Collective ETH/LUSD pool for liquidations
  • Rewards for stakers (ETH gains + LQTY tokens)
LIQUIDITY TOKEN ($LUSD)
  • Algorithmically stabilized by redemptions & arbitrage
GOVERNANCE & INCENTIVES
LQTY Token
  • Staking rewards from fees
  • No governance (protocol is immutable)
Frontend Operators
  • Earn fees for facilitating loans
KEY INNOVATIONS
Recovery Mode
  • Protects system at ETH price crashes (global collateral ratio < 150%)
Redemptions
  • Users swap LUSD for ETH at face value (+ fee)
Gas-Efficient Design
  • Minimal transactions, batch liquidations
ECOSYSTEM PARTNERS
  • **Integrations**: Chainlink, Aave, etc.
  • **Liquid Staking Tokens**: stETH, rETH support
  • **Multi-Chain Expansion**: Layer 2 deployments
