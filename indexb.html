Protocol Suite

Oeconomia Protocol Pantheon

Oeconomia provides a non-custodial, decentralized ecosystem where users retain full control over assets while interacting with five interoperable protocols (Governance, Lending, Liquidity, Trading, and Development Tools).

Alluria Protocol

Alluria enables users to self-custodially mint and manage the ALUD stablecoin using ETH collateral, with a transparent, algorithmically enforced 110% collateral ratio.

Eloqura Protocol

Eloqura offers decentralized tools for liquidity provision, cross-chain transfers, and yield strategies with customizable risk parameters and self-directed optimization.

Artivya Protocol

Artivya is a user-controlled hybrid exchange combining order books and AMMs, with permissionless trading of tokens/NFTs and creator-governed royalty enforcement.

Iridescia Protocol

Iridescia provides public, auditable smart contract templates and security tools for developers to build and deploy their own protocols with community-vetted code libraries.
