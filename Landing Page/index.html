Oeconomia Protocol Pantheon

Oeconomia provides a non-custodial, decentralized ecosystem where users retain full control over assets while interacting with five interoperable protocols (Governance, Lending, Liquidity, Trading, and Development Tools). This integration reduces fragmentation in DeFi by enabling composable, user-directed financial operations without intermediaries.

Alluria Protocol

Alluria enables users to self-custodially mint and manage the ALUD stablecoin using ETH collateral, with a transparent, algorithmically enforced 110% collateral ratio. Borrowers pay a one-time fee (not interest) and retain full control over positions, with liquidations executed by decentralized keepers and stability pools.

Eloqura Protocol

Eloqura offers decentralized tools for liquidity provision, cross-chain transfers, and yield strategies. Users can provide liquidity to AMMs with customizable risk parameters, bridge assets across chains via community-validated, non-custodial systems, and opt into self-directed yield optimization strategies.

Artivya Protocol

Artivya is a user-controlled hybrid exchange combining order books and AMMs, with permissionless trading of tokens/NFTs, creator-governed royalty enforcement, no custody of assets; all settlements occur on-chain.


Iridescia Protocol

Iridescia provides public, auditable smart contract templates and security tools for developers to build and deploy their own protocols. Features include community-vetted code libraries, optional multi-chain deployment, no centralized control over deployments.
